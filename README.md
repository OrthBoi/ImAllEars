# ImAllEars
## Build to help find missing people in Telegram search and rescue channels.
<br>
### How to use:
1. Clone the repo and go into the folder
2. Go into the .env file and put in your custom API ID and HASH
3. Go into the main.py file and under the #CUSTOMIZATION comment put your keywords and a custom session id
4. Right click the folder und click <srg>Open in terminal</srg>
5. Make sure Python is installed and execute <code>Python main.py</code>
6. You may have to verify yourself one more time with a Telegram code
